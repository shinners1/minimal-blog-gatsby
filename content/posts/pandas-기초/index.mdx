#### 어떤 컬럼들이 있는지 확인
```python
data.columns
```

#### 숫자로 되어 있는 컬럼의 평균/중간값 등을 보여줌
```python
data.describe()
```

#### 특정 컬럼 가져오기  
- 열을 가져오게 되면 이 값은 Series로 저장됨 
```python
data['column']
```

#### 두 개 이상의 컬럼 가져오기
- 대괄호가 2번 쓰인다는 것에 주의한다. 
```python
data[['column1', 'column2']]
```

#### 특정 행을 가져오려면
행의 index명으로 가져오는 것은 `loc` - index가 숫자라면 숫자를, 이름이 지정되있다면 해당 이름을 대괄호 안에 써준다.
행의 숫자로 가져오는 것은 `ilok` 
```python
data.loc['행이름 또는 행번호']
data.loc['Seoul']
data.loc[125]
data.iloc[125]
```

#### 두 개 이상의 행을 가져오려면
```python
# 1행과 3행을 가져옴
data.loc[[1,3]]
```

#### 특정 행의 컬럼을 지정해 그 값을 가져오려면
```python
value = data.loc[125, '컬럼명']
```

#### 특정 행의 컬럼을 지정해 거기에 값을 저장하거나 바꾸려면
```python
data.loc[125, '컬럼명'] = value
```

#### 특정 행과 열을 모두 가져오려면
```python
data.loc[[1,3]][['column1','column2']]
```

#### 조건에 맞는 열 값만 보려면 
```python
data.loc[data['column1'] == '2020-08-01', ['column3','column4']]
```

#### 컬럼을 삭제하려면
```python
data = data.drop(['column3','column1'])
```
